# pixi-minesweeper

This is a version of MineSweeper built using the PixiJS framework.

## Code Style

Prettier for formatting Eslint for linting. Some ESLint rules are disabled for early dev

## Testing

Unit tests can be written using Jest. Some libraries are dependant on Webpack loaders and therefor can't be imported to unit test files.
